A class for counted types

A counted type is a type for which an injection to the naturals exists
Class Counted (A : Type) := {
  counted : A → nat;
  counted_inj : ∀ (x y : A), counted x = counted y → x = y;
  incc : ∀ (x : A), nat → A
}.
